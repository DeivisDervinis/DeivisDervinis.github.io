About Me

Hi!

I’m Deivis! I’m a 4th year Computer Science student at Sheridan College currently doing Thesis in Emotion Recognition in Personal Digital Assistants. I am passionate about developing software, particularly in mobile and web.

Some of the projects I’ve worked on include:

Squid Impact: a Space Invaders game clone, written in Java. User interface implemented with JavaX and custom sprites designed in Photoshop.
  • App has 5 levels of difficulty, ability to remember scores and player information
  • Implemented projectile, velocity, and motion algorithms
  • Used pseudorandom generators to display enemies and other obstacles
  • Designed an extensible object-oriented architecture to represent Players, Enemies, Sprites, and Projectiles


Count the Calories! A hybrid cross platform app that enables users to track their weekly grocery purchases and look up calorie counts. Written with HTML5/CSS/Javascript on the Cordova platform.
  • Created data peristance using HTML local storage
  • Designed user interface using JQuery mobile components
  • Integrated Google Charts to displaying calorie counts and nutrition labels
  • Allows users to add products to a cart and see the final total


Pokemon Classic, a final iOS project that allows users to play a classic Pokemon game with original 151 Pokemons. Pokemon Classic includes, Fighting, Pokemon Party, and progress saving using Firebase. Written in Swift for iOS devices.
  • A Classic Pokemon game written in Swift.
  • Contains 1st Generation Pokemon with the use of public web API.
  • Animations with the use of CALayer.
  • Use of Cocoa Pods.
  • Use of Firebase Authentication for user sign up and login.
  • Use of Firebase Firestore Database to store user Pokemons and progress.
  • Use of different gestures for app depth.
